Code Running Instructions

  1. Start SQL: 
    startsql
sql
use 2fa;
  2. Start server.py: 
    python3 server.py
  3. Start ngrok: 
    ngrok tcp 12000
  4. Change TCP URL in loginmanager.py and run: 
    python3 loginmanager.py
cloudflared --url http://localhost:5800
  5. Change the URL in index.html: 
    href = "<cloudflared url>"
